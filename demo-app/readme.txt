/feed : application data in json, it's a replacement for a real RESTful server.
/js : application main file, and util helper
/lib : 3rd party libs
/partials : html partials used in the main page
/index.html : main page of the application

To run the app, just open index.html.


To run it without a server you have to disable security in google chrome.
Simple linux script example :
google-chrome --disable-web-security --allow-access-from-files $*
